# Development Setup

This guide walks you through setting up a local development environment.

## Prerequisites

- Node.js 18+
- npm or pnpm
- Git

## Installation

```bash
git clone https://github.com/RIA-Spec/re-in-act.git
cd re-in-act
npm install
```

## Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the documentation site.

## Quality Checks

```bash
npm run lint
npm run fmt:check
npm run check
```

Use these before opening a pull request.

## Project Structure

<Note>
  The documentation content lives in the `docs/` directory as MDX files. The site is built with
  Next.js and deployed to Cloudflare Workers.
</Note>

| Directory         | Purpose                                   |
| ----------------- | ----------------------------------------- |
| `docs/`           | MDX content and navigation config         |
| `src/app/`        | Next.js pages and layouts                 |
| `src/components/` | Reusable UI components                    |
| `src/lib/`        | Utility functions (MDX compilation, etc.) |

## Build and Preview

```bash
npm run build
npm run preview
```

`preview` runs the OpenNext Cloudflare adapter locally so you can validate deploy behavior before pushing.
